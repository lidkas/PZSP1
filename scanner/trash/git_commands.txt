komendy do gita:

git branch branchname       dodaje gałąź    
git checkout branchname     przełącza się na docelową gałąź     
git add .                   dodaje wszystkie zmienione pliki    
git commit -m "msg"         commit z treścią wiadomości     
git push                    wysyła zmiany      
git pull                    pobiera najnowsze repo, uaktualnia lokalne  
git fetch                   pobiera wybrane pliki
